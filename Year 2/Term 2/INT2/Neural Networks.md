## Neurons

- $x$: input
- $w$: weight ($\theta$)
- $\sum$: Neuron
	- Linear unit, computes $w^{T}x + w_{0}$
- output $z$
- $f$: activation function
	- Non-linear
	- [[]]
	- Linear logistic classifier: sigmoid function
-  Activation $a$

$$a = f(z) = f \left[ \left( \sum\limits^{d}_{j=1} x_{j}w_{j} \right) + w_{0} \right]$$
